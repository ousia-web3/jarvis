# Human Brief 템플릿

인간 대표가 프로젝트를 시작할 때 지휘부와 PM에게 전달하는 최상위 입력 템플릿입니다.

> 사용자가 이 양식을 직접 모두 채울 필요는 없습니다. 사용자가 짧은 자연어 요청을 입력하면 Jarvis와 Friday가 아래 항목을 기준으로 Human Brief 초안을 먼저 자동 작성합니다. 비어 있는 정보는 `추정`, `미정`, `확인 필요`로 표시하고, 위험하거나 되돌리기 어려운 결정만 인간 대표에게 질문합니다.

## 0. 자동 생성 원칙

- 사용자 원문 요청을 그대로 보존한다.
- 목표, 산출물, 성공 기준, 금지 사항을 먼저 추출한다.
- 명확하지 않은 항목은 합리적으로 가정하되 `가정`으로 표시한다.
- 법무, 보안, 개인정보, 외부 공개, 결제, 삭제, 배포처럼 위험도가 높은 항목은 `확인 필요`로 표시한다.
- Human Brief 초안 생성 후 Jarvis 전략 브리프와 Friday 태스크 분해로 자동 진행한다.
