# 아키텍처 폴더

이 폴더는 Jarvis 운영체계의 4단계 아키텍처와 계약을 정의합니다.

## 핵심 문서

- `sys-01-dream-team.md`: 역할 기반 에이전트 팀
- `sys-02-virtual-office.md`: To/CC 커뮤니케이션과 작업 채널
- `sys-03-agent-brain.md`: 기억, 회고, 지혜 승격, 망각 체계
- `sys-04-human-conductor.md`: 인간의 비전, 최종 승인, 리스크 승격
- `contracts.md`: 단계별 입력, 출력, Done 계약
- `operating-principles.md`: 운영 판단 원칙

운영 흐름이 흔들릴 때는 이 폴더를 기준점으로 삼습니다.
