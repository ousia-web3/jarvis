# PRD: 외국인 대상 대한민국 여행정보 콘텐츠 서비스

## 1. 제품 개요

- 제품명: `처음한국 트립 아틀라스` 가칭
- 제품 유형: 공개형 여행정보 콘텐츠 서비스 + 운영자 CMS
- 대상 고객: 외국인 한국 여행 관심자, 외국인 지인을 초대하는 국내 거주자
- 초기 언어: 한국어
- 향후 언어: 영어, 일본어
- 핵심 가치: 외국인의 시선으로 발견한 한국 여행 장면을 실제 지역, 음식, 관광지, 코스로 연결한다.

## 2. 문제 정의

외국인은 유튜브 여행 프로그램에서 한국 음식, 지역, 문화에 흥미를 느끼지만 다음 정보가 흩어져 있다.

- 영상 속 장소가 실제 어느 지역인지
- 메뉴가 어떤 업종/재료/맵기/식사 상황에 해당하는지
- 내 국적, 동행, 여행 스타일에 맞는지
- 서울 외 지역으로 어떻게 확장하면 좋은지
- 실제 이미지는 어떤 기준으로 확인하고 교체해야 하는지

## 3. 사용자 페르소나

| ID | 페르소나 | 목표 | 주요 불안 |
| --- | --- | --- | --- |
| P-01 | 한국 여행 준비 외국인 | 영상에서 본 장소와 음식을 저장하고 일정 후보를 만든다 | 언어, 이동, 메뉴 선택 |
| P-02 | 외국인 친구를 초대하는 한국 거주자 | 국적/동행 유형에 맞는 코스를 제안한다 | 너무 뻔하거나 불편한 코스 |
| P-03 | 가족 여행자 | 아이/부모와 무리 없는 지역과 식당을 고른다 | 이동 피로, 음식 난이도 |
| P-04 | 한류 팬 | 콘텐츠 속 한국을 실제 장소로 방문한다 | 위치 정확도, 운영시간 |
| P-05 | 콘텐츠 운영자 | 영상, 지역, 음식, 관광지 정보를 태깅하고 검수한다 | 저작권, 오래된 정보 |

## 4. 제품 목표

- G-01: 국적, 동행, 관심사 기반으로 한국 여행 콘텐츠를 탐색하게 한다.
- G-02: 지역, 음식, 관광지를 구조화해 검색과 코스화를 가능하게 한다.
- G-03: 유튜브 콘텐츠에서 발생한 관심을 여행정보 수집으로 전환한다.
- G-04: 이미지 권리 확인 전에도 화면 설계를 진행할 수 있는 공통 배너 플레이스홀더 체계를 제공한다.
- G-05: 다국어 확장이 가능한 콘텐츠 모델을 만든다.

## 5. Non-goals

- NG-01: 영상 원본 재호스팅
- NG-02: 권리 확인 없는 썸네일/방송 이미지 사용
- NG-03: 초기 단계에서 실시간 예약/결제 처리
- NG-04: 국적별 성향을 단정적으로 표현
- NG-05: 사용자 개인정보 수집을 기본 전제로 하는 회원 서비스

## 8. 콘텐츠 모델 요구

- 국적/권역: 국가명, 언어, 문화권, 초기 추천 가중치
- 여행객 분류: 동행 유형, 인원 범위, 이동 난이도 민감도
- 지역: 행정구역, 여행권역, 좌표, 대표 테마
- 음식: 업종, 메뉴, 재료, 맵기, 먹는 상황, 주의사항
- 관광지: 유형, 운영시간, 소요시간, 예약 여부, 계절성
- 영상 영감: 유튜브 URL, 에피소드, 제목, 출처, 추출 태그
- 이미지 슬롯: 배너 용도, 상태, 대체 텍스트, 라이선스 메모
- 운영 관리: canonical key, 검수 상태, 품질 점수, 변경 이력, 검색 문서 갱신 상태
- 온톨로지 운영: 클래스, 관계, 근거, 관계 승인 상태, 그래프 품질 진단, 권리 영향 범위
