# IA Brief

## 0. 메타

- requestId: `korea-travel-content-service-planning`
- 프로젝트명: 처음한국 트립 아틀라스
- 서비스 유형: 공개형 여행정보 콘텐츠 서비스 + CMS
- Owner(To): Joi
- CC: TARS, C3PO, Jarvis

## 1. 목표와 범위

- IA가 해결해야 하는 탐색 문제: 외국인 사용자가 국적, 동행 유형, 지역, 음식, 관광지, 영상 영감 중 어느 진입점으로 들어와도 여행정보 상세와 코스 후보에 도달하게 한다.
- 주요 사용자와 핵심 목적 Top 3:
  - 외국인 여행자: 내 취향에 맞는 한국 여행 후보 찾기
  - 한국 거주 초대자: 외국인 친구/가족에게 맞는 코스 추천
  - 운영자: 영상 기반 콘텐츠를 안전하게 태깅하고 관리
- 포함 페이지/섹션: 홈, 탐색, 국적별 추천, 동행별 추천, 지역, 음식, 관광지, 영상 영감, 코스, 검색, 저장함, 가이드, 관리자 CMS
- Non-goals: 예약/결제 UI 확정, 실제 이미지 편집, 회원 정책 확정

| --- | --- | --- | --- | --- |
| L1 | 홈 | 세그먼트 추천과 검색 진입 | P0 | EPIC-1 |
| L1 | 탐색 | 지역/음식/관광지 통합 탐색 | P0 | EPIC-1 |
| L1 | 영상 영감 | 유튜브 콘텐츠 기반 발견 | P0 | EPIC-2 |
| L1 | 코스 | 여행 후보 묶음 | P1 | EPIC-4 |
| L1 | 가이드 | 교통, 주문, 문화 설명 | P1 | EPIC-3 |

| L2 | 상세 페이지 | 지역/음식/관광지/코스 상세 | P0 | REQ-006 |
| L3 | 공통 배너 | 이미지 플레이스홀더 | P0 | REQ-006 |

홈에서 바로 노출할 정보:
- 국적/권역 선택
- 동행 유형 선택
- 인기 음식/지역/영상 영감
- 검색창

2클릭 이내 도달해야 하는 핵심 콘텐츠:
- 국적별 추천 목록
- 동행별 추천 목록
- 음식 상세
- 지역 상세
- 영상에서 본 장소

깊이 제한: 공개 웹은 최대 3단계.

## 4. 내비게이션 모델

| 영역 | 유형 | 항목 | 목적 |
| --- | --- | --- | --- |
| GNB | 글로벌 | 홈, 탐색, 영상 영감, 코스, 가이드 | 주요 섹션 이동 |
| Utility | 유틸 | 검색, 저장함, 언어 | 반복 행동 |
| In-page | 탭 | 개요, 추천 이유, 음식, 관광지, 영상, 이동 | 상세 탐색 |
| Admin LNB | 관리자 로컬 | 콘텐츠, 작업 큐, 이미지, 번역, Trip Ontology | 운영 업무 전환 |
| Breadcrumb | 경로 | 홈 > 지역 > 강원 > 강릉 | 위치 인지 |
| Footer | 보조 | 출처, 권리, 문의, 정책 | 신뢰와 정책 |

탐색 전략: 첫 화면은 `탐색`과 `추천`을 동등하게 둔다. 외국인 사용자는 명확한 목적지가 없을 수 있으므로 국적/동행 기반 추천으로 진입시키고, 지역/음식/관광지 검색으로 보완한다.

## 6. 검색, 필터, 정렬

- 검색 대상: 지역명, 음식명, 메뉴명, 관광지명, 영상 제목, 태그, 국적/권역, 동행 유형
- 필터 축:
  - 국적/권역
  - 동행 유형
  - 지역
  - 음식 업종/메뉴/맵기
  - 관광지 유형
  - 계절
  - 이동 난이도
  - 이미지 상태
- 정렬:
  - 추천순
  - 최신 검수순
  - 영상 인기순
  - 이동 쉬운 순
  - 가족 친화순
- Empty state: "조건을 줄이거나 지역을 넓혀보세요."

## 7. 공통 배너 위치

모든 상세 페이지 상단에는 같은 구조를 둔다.

```text
┌────────────────────────────────────┐
│ [공통 배너 이미지 영역]             │
│ 지역/음식/관광지 이미지 교체 예정   │
│ alt: 콘텐츠 설명 기반 대체 텍스트   │
└────────────────────────────────────┘
```

- 배너 비율: 16:9 PC, 4:3 모바일 허용
- 최소 높이: 모바일 180px, 데스크톱 320px
- 상태 배지: `이미지 준비중`, `공식 이미지`, `직접 촬영`, `공공 라이선스`
- 실제 이미지가 없어도 페이지 제목, 위치, 핵심 태그는 배너 아래에 명확히 표시한다.
