# Human Brief 초안

## 0. 자동 생성 원칙

- 사용자 원문을 기준으로 자동 작성했다.
- 확정되지 않은 사업, 예산, 일정, 권리 조건은 `미정` 또는 `확인 필요`로 표시했다.
- 외부 배포, 저작권 이미지 재사용, 개인정보 수집, 예약/결제 연동은 Human Conductor 승인 전까지 결정하지 않는다.

## 1. 프로젝트 선언

- 프로젝트명: 외국인 대상 대한민국 여행정보 콘텐츠 서비스
- 가칭: `처음한국 트립 아틀라스`
- 한 줄 목표: 외국인이 한국 여행 프로그램에서 본 장소, 음식, 관광 스타일을 국적과 동행 유형별로 탐색하고 여행 후보를 수집하는 콘텐츠 서비스.
- 왜 지금 해야 하는가: 방한 외래관광객 회복과 한류 콘텐츠 영향이 커졌고, 방송/유튜브에서 발견한 여행 욕구를 실제 여행정보 탐색으로 전환할 기회가 있다.
- 최종적으로 얻고 싶은 결과: 전체 서비스 기획 문서, 정보구조, 데이터 모델, 화면 방향, 실행 태스크.

## 2. 비전과 방향성

- 핵심 문제: 외국인은 한국 여행 콘텐츠를 보며 흥미를 느끼지만, 영상 속 지역, 음식, 이동, 동행 유형별 적합성 정보를 한곳에서 구조적으로 찾기 어렵다.
- 타깃 사용자: 한국 여행에 관심 있는 외국인, 외국인 친구/가족을 한국으로 초대하는 국내 거주자, 여행 콘텐츠 편집자.
- 사용자가 느껴야 할 변화: "영상에서 본 한국"이 "내 여행 후보"로 정리된다.
- 브랜드/톤/철학: 신뢰 가능한 여행 큐레이션, 친절한 한국어, 외국인의 첫 시선을 존중하는 설명.

## 3. 성공 기준

- 80점 기준:
  - 국적, 동행 유형, 지역, 음식, 관광지 축으로 콘텐츠를 탐색할 수 있다.
  - 상세 페이지마다 공통 이미지 배너 플레이스홀더가 있다.
  - 유튜브 콘텐츠 분석 근거와 저작권 리스크가 문서화되어 있다.
- 100점 기준:
  - 여행객 국적/분류별 추천 로직, 다국어 확장 모델, CMS 운영, 지도/검색/분석 체계까지 연결된다.
  - 실제 이미지 교체 전까지 모든 시각 영역이 표준 플레이스홀더로 일관되게 표시된다.
  - 콘텐츠 출처, 저작권 상태, 검수 상태를 데이터 모델에서 추적한다.
- 반드시 포함:
  - 여행객 국적별 선호 지역/관광 스타일
  - 여행객 분류별 선호 지역/관광 스타일
  - 지역, 음식정보, 관광지 데이터 구조
  - 영상 기반 콘텐츠 태깅 체계
  - 공통 배너 이미지 박스 규칙
- 절대 포함하지 않음:
  - 권리 확인 없는 유튜브 썸네일/방송 이미지 재사용
  - 실제 예약/결제 처리 확정
  - 외국인 개인 데이터 수집 확정

## 4. 제약 조건

- 시간: 미정
- 예산: 미정
- 사용 가능한 도구: 공개 웹 자료, 로컬 문서, 유튜브 공개 메타데이터, Jarvis 운영 스크립트
- 사용 금지 도구: 권한 없는 영상 다운로드, 무단 이미지 사용, 개인정보 수집 자동화
- 법무/보안/개인정보 제약:
  - 영상은 분석과 링크 참조까지만 사용한다.
  - 이미지 영역은 플레이스홀더로 표기하고, 실제 이미지는 추후 라이선스 확인 후 교체한다.
  - 회원, 즐겨찾기, 후기 기능은 개인정보보호 설계가 필요하다.

## 6. 최종 승인 조건

- 승인자: Human Conductor
- 승인 전 필수 검토:
  - 서비스명과 브랜드 톤
  - 우선 지원 언어
  - 유튜브/방송 콘텐츠 활용 권리 범위
  - 지도, 예약, 제휴, 회원 기능 범위
- 배포 또는 외부 공유 가능 여부: 확인 필요
- 추가 코칭 메모: 이번 문서는 전체 서비스 설계 초안이며, MVP 범위로 축소하지 않는다.
