## 6. 다양성 QA 규칙

- 단일 Cat ID 사용 이미지는 전체 86장의 12% 이하로 유지한다.
- 같은 목록에서 동일 Cat ID가 연속 2장 이상 반복되지 않게 한다.
- CAT-13은 담요·온도 맥락을 필요한 장면에만 사용하고 주름을 희화화하지 않는다.
- CAT-14는 스코티시 **스트레이트**로 고정하고 접힌 귀를 생성하지 않는다.
- 품종명을 제목·배지·행동 설명에 자동 노출하지 않는다.
- 다묘 장면은 각 고양이의 실루엣, 네 발, 한 개의 꼬리가 분리되어야 한다.
