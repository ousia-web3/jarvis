# Jarvis 전략 브리프

## 전략 판단

이번 작업의 중심은 “유튜브 분석 결과를 책으로 옮기는 것”이 아니라, 고양이 보호자의 기억에 남는 독서 경험으로 재설계하는 것이다. 원천 콘텐츠는 수의학적 신뢰와 생활 관찰의 방향성을 얻는 리서치 자료로만 쓰고, `냥냥츄르`는 별도의 세계관과 인물, 장면, 독서 인터랙션을 갖는 독립 창작물로 만든다.

## 핵심 제품 정의

- 제품명: 냥냥츄르
- 형식: 빠르게 읽는 고양이 이야기책 + 보호자 메모 + 장면별 체크 카드
- 독서 리듬: 3분 에피소드, 30초 체크, 1문장 다짐
- 주인공: 길에서 온 듯하지만 집의 규칙을 다시 가르치는 고양이 `츄르`
- 독자 역할: 구경꾼이 아니라 이야기 속 집사 훈련생

## Agent Assignment Preview

- 요청 요약: 미야옹철의 냥냥펀치 비-Shorts 전체 분석 기반 `냥냥츄르` 책 원고와 소개 웹페이지 제작
- To: EVE(리서치), Data(분석 검증), C3PO(원고), Joi(독서 UX/웹), TARS(웹 구현), Jarvis/Friday(전략/운영)
- CC: KITT/TRON(저작권·의료·AI 탐지 회피 리스크), Diagnostic Agent(완료 근거 점검)
- Risk: Medium. 공개 콘텐츠·의료 표현·저작권·탐지 회피 요청 변환 이슈 존재
- Expected Outputs: 리서치 리포트, 문체 스킬, 책 기획서, 원고, 소개 웹페이지, 이미지 자산, 검증 증거
- 다음 액션: 자막 기반 분석 완료 후 책 구조와 원고에 반영

## 성공 원칙

- 모든 인사이트는 “보호자가 관찰할 수 있는 장면”으로 바꾼다.
- 병명/치료법 단정은 피하고 “관찰 기록, 위험 신호, 병원 상담”으로 표현한다.
- 고양이를 인간 감정의 소품으로 쓰지 않고, 고양이의 감각과 경계, 선택권을 존중한다.
- 빠른 독서를 위해 긴 장 대신 짧은 에피소드와 반복 가능한 페이지 구조를 쓴다.
