# Human Brief 초안

## 1. 프로젝트 선언

- 프로젝트명: 냥냥츄르 고양이 이야기 책과 소개 웹페이지
- 한 줄 목표: 미야옹철의 냥냥펀치 채널의 고양이 돌봄/행동/건강 콘텐츠를 분석해, 일반 독자가 빠르게 읽을 수 있는 새로운 형식의 고양이 이야기 책과 소개 웹페이지를 만든다.
- 왜 지금 해야 하는가: 고양이 보호자에게 필요한 지식은 많지만, 영상 기반 정보는 흩어져 있고 빠르게 다시 읽기 어렵다. 이를 감정 있는 짧은 이야기와 실용 메모가 결합된 책으로 재구성하면 접근성이 높아진다.
- 최종적으로 얻고 싶은 결과: 리서치 리포트, 출판 가능한 책 기획/원고 초안, 프로젝트 전용 문체 스킬, 이미지 자산, 완성된 소개 웹페이지.

## 2. 비전과 방향성

- 핵심 문제: 고양이 정보가 지식으로는 유용하지만 독자가 오래 기억할 만큼 이야기화되어 있지 않다.
- 타깃 사용자: 고양이를 키우는 보호자, 예비 집사, 고양이 행동과 건강을 부담 없이 배우고 싶은 일반 독자.
- 사용자가 느껴야 할 변화: “정보를 공부했다”보다 “고양이를 조금 더 잘 이해하게 됐다”는 감각.
- 브랜드/톤/철학: 다정하지만 과장하지 않고, 귀엽지만 유치하지 않으며, 의학 정보는 단정 대신 관찰과 상담으로 안내한다.

## 3. 성공 기준

- 80점 기준: Shorts 제외 영상 목록과 접근 가능한 자막을 분석하고, 주제 분류/책 구조/원고 일부/웹페이지를 완성한다.
- 100점 기준: 전체 비-Shorts 자막 기반 신호를 반영해 책 전체 원고 초안을 완성하고, 웹페이지는 이미지와 반응형 레이아웃까지 검증한다.
- 반드시 포함: 채널 전체 메타데이터 분석, 자막 기반 비원문 분석, 새로운 독서 형식, `냥냥츄르` 독립 창작 서사, 웹 소개 페이지.
- 절대 포함 금지: 영상 대본 장문 복제, 원작 말투의 노골적 모방, AI 탐지 회피 목적의 지침, 수의학적 진단 단정.

## 4. 제약 조건

- 시간: 사용자는 제한 없음. 현재 턴 안에서 가능한 범위까지 멈추지 않고 진행.
- 예산: 별도 예산 없음.
- 사용 가능한 도구: yt-dlp, youtube-transcript-api, 이미지 생성 스킬, 로컬 웹 구현/검증.
- 사용 금지/주의 도구: 외부 배포, 비밀키 요구, 유튜브 원문 대량 재배포.
- 법무/보안/개인정보 제약: 공개 콘텐츠 분석만 사용. 자막 원문은 저장하지 않고 분석 신호만 저장.
