# 30초 체크 동적 기능 검증

## 요청 요약

- `manuscript.html`의 `30초 체크` 항목을 일반 독자가 직접 체크할 수 있게 개선한다.

## 구현 내용

- 각 장의 `30초 체크` 목록을 페이지 로드 시 체크박스 UI로 변환했다.
- 장별 진행률을 `0/4 완료`처럼 표시하고, 체크 시 진행 막대가 함께 변한다.
- 체크 상태는 브라우저 `localStorage`에 저장되어 새로고침 후에도 유지된다.
- 장별 `초기화` 버튼으로 해당 장의 체크 상태를 지울 수 있다.
- 저장 값은 체크 여부와 항목 식별자뿐이며 외부 전송은 없다.

## 검증 결과

- 체크 섹션 수: 42개
- 생성된 체크박스 수: 168개
- 첫 장 체크 전: `0/4 완료`
- 첫 장 1개 체크 후: `1/4 완료`
- 새로고침 후 유지: `1/4 완료`, 체크 상태 유지됨
- 초기화 후: `0/4 완료`, 체크 해제됨
- 모바일 390px 폭 가로 오버플로: 없음
