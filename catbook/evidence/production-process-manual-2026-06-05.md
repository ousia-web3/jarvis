# 제작 프로세스 매뉴얼 HTML 생성 검증

## 요청 요약

- 대상: `catbook/`
- 산출물: `catbook/web/production-process-manual.html`
- 목적: 프로젝트 폴더 분석 결과를 바탕으로 제작 프로세스 매뉴얼을 단일 HTML 파일로 제공한다.

## 반영 내용

- CSS와 JavaScript를 모두 HTML 안에 인라인으로 포함했다.
- 히어로 이미지는 원본 PNG 대신 최적화 JPG 파생본을 호출한다.
- 모바일은 `hero-window-cat-768.jpg`, 태블릿/데스크톱은 `hero-window-cat-1280.jpg`를 media 조건으로 분기한다.
- 리서치, 문체 설계, 원고 조립, 이미지 자산, 웹 빌드, 검증 단계 7개를 동적 카드로 구성했다.
- 검색, 역할 필터, 단계 세부 모달, 산출물 경로 복사, 운영 체크리스트, 로컬 저장 진행률을 구현했다.
- 산출물 20개와 주요 실행 명령을 매뉴얼 안에서 바로 확인하고 복사할 수 있게 했다.
