# 에이전트 지침 폴더

이 폴더는 Jarvis Dream Team의 역할별 지침을 보관합니다.

## 사용 순서

1. 역할 배정 기준은 `00-agent-management-index.md`에서 확인합니다.
2. Owner(To)와 CC를 정한 뒤 해당 에이전트 파일을 읽습니다.
3. 새 에이전트를 추가하면 인덱스, `docs/project-user-manual.html`, `docs/README.md`를 함께 갱신합니다.

## 표기 원칙

- 디자인과 UX 역할은 `Joi` 단일 표기를 사용합니다.
- 과거 병기 표기는 신규 문서에 사용하지 않습니다.
