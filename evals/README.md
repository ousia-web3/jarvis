# Jarvis 평가 하네스

이 폴더에는 에이전트 행동을 점수화하기 위한 평가 과제와 루브릭이 들어 있습니다.

## 사용 방법

1. `evals/tasks/`에서 평가 과제를 고른다.
2. 일반적인 시작/검증/완료 스크립트를 사용해 Jarvis 워크플로를 실행한다.
3. `evals/rubrics/`의 대응 루브릭으로 결과를 채점한다.
4. 발견한 격차는 지혜 후보 또는 에이전트 카드 개선 항목으로 기록한다.

## 점수 구간

- 0-5: 역할 혼선 또는 검증 누락
- 6-7: 사용할 수 있으나 수동 지도가 많이 필요함
- 8-9: 명확한 게이트가 있어 안정적으로 반복 가능함
- 10: 제품 수준으로 반복 가능하고, 검증과 기억 체계까지 연결됨
