# 지혜 승격 절차

## 목적

에이전트의 일회성 경험을 반복 가능한 운영 원칙으로 승격합니다.

## 단계

1. Raw Log 수집
2. Episodic Memory 작성
3. Data가 반복 패턴 식별
4. Jarvis가 공통 원리 후보 선별
5. Friday가 운영 규칙으로 변환
6. KITT/TRON이 리스크 있는 규칙을 검토
7. 승인된 원칙을 지혜 저장소에 등록

## 지혜 후보 기준

- 두 번 이상 반복된 성공 패턴
- 반복 실패를 줄일 수 있는 규칙
- 다른 프로젝트에도 적용 가능한 원리
- 비용, 시간, 품질, 안전성을 개선하는 절차
