# PDF 추출 보존 자료

이 폴더는 별도 보관된 `AI_에이전트_팀_아키텍처.pdf`에서 추출한 Markdown, JSON, 이미지 파일을 보존하는 자료실입니다. 프로젝트 내부의 원본 PDF는 2026-05-22에 사용자 승인으로 삭제했습니다.

## 사용 원칙

- 이 폴더는 살아있는 운영 문서가 아니라 외부 보관 원본의 추출 결과 보존 위치입니다.
- 일상적인 설명, 최신 운영 기준, 사용자 안내는 상위 `docs/`의 문서에 작성합니다.
- 이미지와 추출 Markdown은 매뉴얼이나 보고서에서 근거 자료로 참조할 수 있습니다.
- 파일 이동, 삭제, 대량 이름 변경은 링크 영향도가 크므로 Human Conductor 승인 후 별도 작업으로 진행합니다.

## 대표 파일

- `AI_에이전트_팀_아키텍처.md`: PDF에서 추출한 Markdown
- `AI_에이전트_팀_아키텍처.json`: 추출 메타데이터
- `images/`: PDF 페이지와 도식에서 추출된 이미지
