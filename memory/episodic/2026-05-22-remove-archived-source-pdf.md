# 에피소딕 메모리: 별도 보관된 원본 PDF 삭제

## 상황

사용자는 `docs/AI_에이전트_팀_아키텍처.pdf`를 별도 보관하고 있으니 프로젝트 안에서 필요 없다면 삭제하자고 요청했다.

## 판단

매뉴얼과 운영 문서는 원본 PDF 자체보다 `docs/opendataloader-extract/`의 이미지, Markdown, JSON을 참조하고 있었다. 따라서 원본 PDF는 삭제해도 되고, 추출 보존 자료는 남기는 것이 맞다고 판단했다.

## 결과

- 원본 PDF를 프로젝트에서 삭제했다.
- 활성 문서는 “원본 PDF는 별도 보관, 프로젝트 내부는 추출물 보존” 상태로 정리했다.
- 추출물과 매뉴얼은 계속 사용할 수 있음을 확인했다.

## 재사용할 교훈

대형 원본 파일은 외부에 안정적으로 보관되어 있고 프로젝트 안에서는 추출물만 필요하다면, 프로젝트에는 추출물과 출처 설명만 남기는 편이 가볍다.
