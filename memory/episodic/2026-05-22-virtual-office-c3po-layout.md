# Episodic Memory: Virtual Office C3PO 배치 개선

## 상황

Virtual Office에서 C3PO 캐릭터가 우상단 포커스 패널에 가려지는 문제가 보고되었다. 사용자는 포커스 패널을 팝업보다 전광판으로 보고, 에이전트 위치와 배경을 함께 바꾸는 방향을 제안했다.

## 판단

문제는 C3PO의 `copy` 스테이션이 `x: 71, y: 20`으로 우상단 패널 영역과 겹치는 좌표에 있었기 때문이다. 포커스 패널을 완전히 제거하기보다 사무실 배경 속 전광판으로 의미를 바꾸고, C3PO를 하단 커뮤니케이션 테이블 쪽으로 이동하는 편이 Virtual Office 은유를 더 잘 보존한다.

## 결과

- 배경을 `virtual-office.png`로 교체했다.
- 포커스 패널을 어두운 전광판 스타일로 조정했다.
- C3PO 위치를 `x: 57, y: 83`으로 이동했다.
- 좁은 화면과 데스크톱 화면에서 C3PO와 패널 겹침 면적 `0`을 확인했다.

## 다음에 참고할 점

Virtual Office의 전면 패널은 독립 팝업처럼 보이면 캐릭터와 충돌한다. 상단 우측은 전광판 존으로 예약하고, 커뮤니케이션/문서화 역할 에이전트는 하단 테이블이나 사이드 데스크에 배치하는 것이 안전하다.
