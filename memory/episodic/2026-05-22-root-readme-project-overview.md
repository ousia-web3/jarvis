# 에피소딕 메모리: 루트 README 프로젝트 개요 작성

## 상황

사용자는 Jarvis 프로젝트 전체를 설명하는 루트 README가 필요하다고 지적했다. 단, 주식 하위 프로젝트와 오늘의할일 계열 폴더는 개요에서 제외하길 원했다.

## 판단

`docs/README.md`는 내부 문서 허브 역할을 하고 있었지만, 프로젝트 루트에 처음 들어온 사람이 Jarvis 운영체계를 바로 이해할 수 있는 첫 안내서는 없었다. 따라서 루트 `README.md`를 별도로 만들고, `docs/README.md`에서 루트 개요를 연결하는 방식이 가장 자연스럽다고 판단했다.

## 결과

- 루트 `README.md`를 생성했다.
- `docs/README.md`의 시작점에 루트 프로젝트 개요 링크를 추가했다.
- README 범위에서 `stock-auto-trader/`와 오늘의할일 계열 폴더를 제외한다고 명시했다.

## 재사용할 교훈

Jarvis 프로젝트에 새 루트 폴더나 핵심 운영 문서가 추가될 때는 `README.md`와 `docs/README.md`를 함께 갱신해야 문서 탐색 경로가 끊기지 않는다.
