# Agent Brain 기억 저장소

이 폴더는 SYS.03 Agent Brain의 작업 저장소입니다.

## 폴더와 파일

- `work-logs/`: 작업 중 또는 작업 완료 후 생성되는 구조화 실행 로그
- `episodic/`: 작업 단위 회고 기억
- `wisdom-candidates.md`: 반복 패턴을 지혜로 승격하기 전 보관하는 후보 목록
- `wisdom-registry.md`: 승인된 재사용 운영 원칙
- `purge-queue.md`: 요약 또는 제거를 기다리는 원천 자료와 중복 자료
- `quarantine.md`: 드리프트, 보안 우려, 의심스러운 완료 보고를 검토 대기 상태로 보관하는 목록

## 정책

- 최종 산출물, 결정, 승인된 지혜, 검증 증거를 보존한다.
- 원천 로그와 리서치 자료가 잡음이 되면 요약한다.
- 비밀키, 계좌 정보, 개인정보, 실거래 주문 세부값은 저장하지 않는다.
- Human Conductor의 명시 승인 없이 자동화로 사용자 파일을 삭제하지 않는다.
